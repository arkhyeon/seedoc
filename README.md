# SeeDoc
> 병원예약 및 병원 개인 페이지 제작 서비스

## 개발 목표
- 2020년 06월 22일 ~ 2020년 07월 21일  종료
1. 병원 사전 예약으로 대기 시간 감소
2. 환자에게 맞는 병원 / 의사 찾기
3. 병원 이용자만 작성 할 수 있는 믿을 만한 리뷰
4. 캘린더를 통한 예약 System 구축
5. 지도api를 통한 병원 찾기 구현
6. 병원에서 직접 관리하는 건강상식 게시판
